Hit & Blow

ヒットアンドブロー

コンピューターが決めた3桁の数字を当てるゲームです。

  1. 百の位、十の位、一の位にそれぞれ0から9を入れます。
  2. 数字と位置一致したらイート(eat)、数字のみ一致したらバイト(bite)として、それぞれの個数がわかります。
  3. ただし、百の位は0を除き、数字の重複も除きます。
  • 作成期間
    • 1週間ほどで作成しました。
  • 使用技術
    • Java
  • 学んだこと、苦労したこと
    • 学んだことは、メソッド関連です。
    苦労したことは、元々C言語で作成していたものをどのようにしてJavaの仕様にあったプログラミングを行っていくことです。
